# Interactive Login Form with JavaScript

## 📌 Overview
This project is a simple interactive login form built using HTML, CSS, and JavaScript.

When a user enters their name and password and clicks the Sign In button, the application displays a personalized verification message.

## 🚀 Features
- Username Input
- Password Input
- Sign In Button
- Dynamic Welcome Message
- Clean User Interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📷 Output
Displays a personalized message such as:

Hi Priti, Verifying Your Account...
